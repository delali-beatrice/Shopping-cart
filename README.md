# 🛒 JavaScript Shopping Cart

A dynamic shopping cart web app built with vanilla HTML, CSS and JavaScript — featuring product listings, add/remove functionality and a live updating total.

## ✨ Features

- 🖼️ Product grid dynamically rendered from a JavaScript array
- ➕ Add items to cart with quantity tracking
- ➖ Remove items from cart with validation
- 💰 Live updating cart total
- 📱 Responsive grid layout
- 🎨 Clean and minimal UI

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript (no frameworks or libraries)

---

## 📁 Project Structure

```
shopping-cart/
```
